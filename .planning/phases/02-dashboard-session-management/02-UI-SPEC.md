## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px | 400 (regular) | 1.5 |
| Label | 12px | 500 (medium) | 1.4 |
| Heading | 20px | 600 (semibold) | 1.2 |
| Display | 28px | 700 (bold) | 1.2 |

Notes:
- Display used only for the stats bar numbers (total sessions, average score, best score)
- Heading used for section titles ("Past Sessions", "Start a New Interview")
- Label used for badges, timestamps, and metadata on session cards
- Body used for all other text

## Component Inventory

### 1. Stats Bar

Horizontal bar with three stat cells. Placed above session list.

| Stat | Label | Value Format |
|------|-------|-------------|
| Total Sessions | "Sessions" | Integer (e.g. "12") |
| Average Score | "Avg Score" | Integer with "/100" suffix (e.g. "74/100") |
| Best Score | "Best" | Integer with "/100" suffix (e.g. "92/100") |

Layout: `flex items-center justify-between` inside a `bg-gray-900 border border-gray-800 rounded-xl p-4` container. On mobile (below 768px), stack vertically with `gap-3`.

When user has no scored sessions: show "---" for Average and Best, "0" for Sessions.

### 2. Session Card

Compact card for each past interview session.

| Element | Spec |
|---------|------|
| Container | `rounded-xl border border-gray-800/60 bg-gray-900/40 p-4` |
| Question title | 14px, weight 400, `text-gray-100`, truncate with `line-clamp-1` |
| Difficulty badge | Reuse `PromptBadge` component (easy=emerald, medium=amber, hard=red) |
| Score | 20px, weight 600, color by range (emerald/amber/red), or "Not scored" in `text-gray-500` italic |
| Date | 12px, weight 400, `text-gray-500`, relative format ("2 days ago") |
| Retake button | `text-sm font-medium text-blue-400 hover:text-blue-300` link-style, right-aligned |

Layout: Row on desktop (`flex items-center justify-between gap-4`). On mobile, two rows: title+badge top, score+date+retake bottom.

### 3. Prompt Picker Grid

Reuse existing `PromptCard` pattern from `src/app/page.tsx`.

Layout: `grid grid-cols-1 sm:grid-cols-2 lg:grid-cols-3 gap-4`

### 4. Mobile Canvas Notice

Shown on `/session/[id]` when viewport < 768px.

| Element | Spec |
|---------|------|
| Container | `bg-amber-900/30 border border-amber-700/50 rounded-lg p-4 mx-4 mt-4` |
| Icon | `Monitor` from lucide-react, 20px, `text-amber-300` |
| Text | "This experience works best on desktop. Switch to a larger screen for the full canvas." 14px, `text-amber-200` |

## Interaction States

| Component | State | Behavior |
|-----------|-------|----------|
| Session card | Hover | `border-gray-700` transition, subtle lift not needed (flat dark UI) |
| Session card | Loading | Skeleton pulse (`animate-pulse`) matching card dimensions |
| Retake button | Hover | `text-blue-300` from `text-blue-400` |
| Start Interview button | Default | `bg-blue-600 text-white rounded-lg px-4 py-2` |
| Start Interview button | Hover | `bg-blue-500` |
| Start Interview button | Loading | Spinner icon replacing text, button disabled |
| Stats bar | Loading | Three skeleton rectangles with `animate-pulse` |
| Empty state | Default | Centered illustration-free layout, heading + body + arrow pointing down to prompt grid |
